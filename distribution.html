How to Buy Hot Potatoes (Recommended Configuration)
  1. Download the Metamask Chrome extension here.
  2. Run Metamask.
  3. Create a new wallet.
  4. Transfer some Ether to the Metamask wallet.
  5. Buy hot potatoes!

Alternatively, you can directly transfer Ether to the wallet contract address.
